# 后端运行环境配置

## 1. 环境要求

### 1.1 Python版本
- Python 3.7+（推荐Python 3.9）

### 1.2 操作系统
- Windows
- Linux
- macOS

## 2. 依赖包

### 2.1 核心依赖
```
fastapi
uvicorn
python-multipart
Pillow
numpy
ultralytics
opencv-python
```

### 2.2 依赖说明
- **fastapi**: Web框架，用于构建API接口
- **uvicorn**: ASGI服务器，用于运行FastAPI应用
- **python-multipart**: 用于处理文件上传
- **Pillow**: 图像处理库
- **numpy**: 数值计算库
- **ultralytics**: YOLO模型库，用于加载和运行火焰检测模型
- **opencv-python**: 计算机视觉库，用于图像处理和视频分析

## 3. 安装步骤

### 3.1 安装Python
从[Python官网](https://www.python.org/downloads/)下载并安装Python 3.7或更高版本。

### 3.2 安装依赖
在项目根目录下执行以下命令：

```bash
pip install -r requirements.txt
```

### 3.3 验证安装
执行以下命令验证依赖是否安装成功：

```bash
pip list | grep -E "fastapi|uvicorn|python-multipart|Pillow|numpy|ultralytics|opencv-python"
```

## 4. 运行方法

### 4.1 启动服务
在项目根目录下执行以下命令：

```bash
python main.py
```

### 4.2 验证服务
服务启动后，可以通过以下方式验证：
- 访问根路径：`http://localhost:8000`
- 访问API文档：`http://localhost:8000/docs`

## 5. 模型文件

### 5.1 模型要求
项目需要 `yolo10n_fire.pt` 模型文件，该模型基于YOLOv10n和自定义数据集训练，用于火焰检测。

### 5.2 模型位置
模型文件应放置在项目根目录下：
```
ForestFire/Back/yolo10n_fire.pt
```

## 6. 目录结构

```
ForestFire/Back/
├── app/
│   ├── api/            # API路由
│   ├── services/       # 业务逻辑
│   └── __init__.py
├── config/            # 配置文件
├── docs/              # 文档
├── uploads/           # 临时文件存储
├── main.py            # 应用入口
├── requirements.txt   # 依赖文件
└── yolo10n_fire.pt   # 火焰检测模型
```

## 7. 环境变量（可选）

如果需要配置环境变量，可以在项目根目录创建 `.env` 文件，例如：

```
# .env
API_V1_STR=/api
PROJECT_NAME=ForestFire Backend
VERSION=1.0.0
```

## 8. 注意事项

- 确保 `yolo10n_fire.pt` 模型文件存在且可访问
- 确保 `uploads/` 目录存在且有写入权限
- 对于视频分析，处理时间可能较长，建议限制视频大小和长度
- 在生产环境中，建议配置CORS和安全设置

## 9. 故障排除

### 9.1 模型加载失败
- 检查 `yolo10n_fire.pt` 文件是否存在
- 检查Ultralytics库版本是否兼容

### 9.2 依赖安装失败
- 尝试使用 `pip install --upgrade pip` 升级pip
- 对于Windows用户，可能需要安装Visual C++ Build Tools

### 9.3 服务启动失败
- 检查端口8000是否被占用
- 检查Python版本是否符合要求
- 检查依赖是否安装完整

## 10. 部署建议

### 10.1 开发环境
- 使用 `python main.py` 直接运行
- 启用热重载，便于开发调试

### 10.2 生产环境
- 使用 `uvicorn main:app --host 0.0.0.0 --port 8000 --workers 4` 运行
- 配置反向代理（如Nginx）
- 启用HTTPS
- 设置合适的文件上传限制
